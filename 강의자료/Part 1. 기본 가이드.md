# 옵시디언의 특징
1. 모든 노트는 로컬 디바이스에 저장됨 (속도, 보안 강점)
2. 그래프 뷰, 양방향 링크 [[환영합니다]]
3. [[마크다운 문법]]
4. 웹 뷰어 
	- 코어 플러그인 > 웹 뷰어 활성화
		- 옵시디언에서 웹 링크 실행 및 보관함 저장
		- 크롬 확장 프로그램 지원
		- https://naver.com
5. 커뮤니티 플러그인
6. 무료 (상업적 이용 포함)
7. 장단점
	- (장점) 개인의 기록 습관에 맞게 노트 관리 프로세스 체계화 가능
	- (단점) 높은 자유도로 인한 노트 관리의 어려움

# Features of Obsidian
1. All notes are stored on your local device (advantages: speed, security)
2. Graph view, bidirectional links [[Welcome]]
3. [[Markdown Syntax]]
4. Web Viewer
   - Core Plugin > Enable Web Viewer
     - Run web links and save them in your vault within Obsidian
     - Chrome extension supported
     - https://naver.com
5. Community plugins
6. Free (including commercial use)
7. Pros and Cons
   - (Pros) Can systematize your note management process according to your personal recording habits
   - (Cons) Difficult to manage notes due to high degree of freedom

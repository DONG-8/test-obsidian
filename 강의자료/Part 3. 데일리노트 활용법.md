# 데일리노트 활성화
- 설정 > 코어 플러그인 > 일일 노트
	- 앱 시작 시 일일 노트 열기 활성화

# 실습
1. Calendar 플러그인
	- [[일일 기록 업데이트]]
	- 템플릿 노트 생성 및 연결
		- 설정 > 일일 노트 > 템플릿 파일 위치
	- Calendar 플러그인 - 주간 기록 활성화
		- Calendar > Show week number 활성화
2. To-Do 리스트
	- [[구글 캘린더 일정 가져오기]]
3. Templater 플러그인
4. Outliner 플러그인
	- 블렛 노트 (단축키 및 마우스 이동 지원)
		- 위/아래 이동
			- **Windows:** Ctrl + Shift + ↑ or ↓
		    - **macOS:** Command(⌘) + Shift(⇧) + ↑ or ↓
		- 접기/펼치기
			- **Windows:** Ctrl + ↑ or ↓
			- **macOS:** Command(⌘) + ↑ or ↓
5. ChatGPT로 주간 회고 자동 요약
6. Journal Review 플러그인
7. [[단축키]]

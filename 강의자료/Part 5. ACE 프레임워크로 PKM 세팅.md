# ACE 프레임워크
닉 마일로의 ACE 프레임워크에서 구조만 가져왔으며, 저에게 맞는 버전으로 커스텀한 구조입니다.

| 1depth     | 2depth         | 설명                                   |
| ---------- | -------------- | ------------------------------------ |
| `+`        |                | Inbox, 정리되지 않은 노트                    |
|            | `Clippings`    | 웹 뷰어, 노트 수집                          |
| `6F`       |                | 컬렉션 모음 (내 삶의 우선순위 시각화)               |
| `Atlas`    |                | 정보&지식 기록 (시간과 관계 없는 기록)              |
|            | `Maps`         | 주제 중심의 노트 연결 모음                      |
|            | `Notes`        | 일반 모든 노트                             |
|            | `Visuals`      | 시각화 노트                               |
|            | `Books`        | 도서 노트                                |
| `Calendar` |                | 시간과 연결된 기록                           |
|            | `Days`         | 데일리노트                                |
|            | `Logs`         | 주제 중심의 연속성 기록                        |
|            | `Records`      | 단발성 기록                               |
| `Efforts`  |                | 목표, 노력과 연결된 기록                       |
|            | `On`           | 진행중인 프로젝트                            |
|            | `Ongoing`      | 반복되는 프로젝트                            |
|            | `Simmering`    | 아이디어 구상 단계의 프로젝트                     |
|            | `Sleeping`     | 잠시 정지된 프로젝트                          |
|            | `Archive`      | 완료한 프로젝트                             |
| `Kit`      |                | 노트 관리에 필요한 도구 모음                     |
|            | `Attachments`  | 첨부파일                                 |
|            | `DB&Structure` | 노트 컬렉션을 위한 DB Structure              |
|            | `Excalidraw`   | Excalidraw에서 사용하는 script 등의 유틸 파일 모음 |
|            | `Templates`    | 템플릿                                  |

# 실습
- 폴더 생성
- Calendar > Days 폴더
	- 데일리노트 파일 이동
	- 설정 > 일일 노트 > 새 파일 위치 > Calendar/Days
- Kit > Attachments
	- 첨부파일 이동
	- 설정 > 파일 및 링크 > 새 첨부 파일을 생성할 위치 > 아래에 지정된 폴더 > Kit/Attachements
- Atlas > Visuals
	- 시각화 파일 이동(eacalidraw, canvas)
	- 설정 > Excalidraw > Basic > Excalidraw folder > Atlas/Visuals
- Kit > Templates
	- Dailynote 템플릿 파일 이동
	- 설정 > 일일 노트 > 템플릿 파일 위치 > Kit/Templates/Dailynote
	- 설정 > Templater > Kit/Templates
- Atlas > Notes
	- 모든 노트 파일 이동
- Atlas > Books (옵션)
	- 도서 노트 파일 이동
- Clippings 폴더 이동
- 강의자료 폴더 생성 (임시)
- 인박스 설정
	- 설정 > 파일 및 링크 > 새 노트를 생성하는 위치 > 아래에 지정된 폴더 > +

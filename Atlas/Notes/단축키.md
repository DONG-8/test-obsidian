## 단축키 설정 경로
- 설정 > 단축키

| 실행                  | Windows                           | macOS                                        |
| ------------------- | --------------------------------- | -------------------------------------------- |
| 일일 노트: 오늘의 일일 노트 열기 | Ctrl + Shift + D                  | Control(⌃) + Shift(⇧) + D                    |
| 일일 노트: 내일의 일일 노트 열기 | Ctrl + Shift + .                  | Control(⌃) + Shift(⇧) + .                    |
| 일일 노트: 어제의 일일 노트 열기 | Ctrl + Shift + ,                  | Control(⌃) + Shift(⇧) + ,                    |
| 오른쪽 사이드바 열기/닫기      | Ctrl + Shift + ]                  | Command(⌘) + Shift(⇧) + ]                    |
| 왼쪽 사이드바 열기/닫기       | Ctrl + Shift + [                  | Command(⌘) + Shift(⇧) + [                    |
| 제목 설정               | Ctrl + Alt + 1 <br>`제목 크기별 숫자 설정` | Command(⌘) + Option(⌥) + 1<br>`제목 크기별 숫자 설정` |
| 뒤로가기                |                                   | Command(⌘) + [                               |
| 앞으로가기               |                                   | Command(⌘) + ]                               |
| 다른 폴더로 파일 이동        | Ctrl + Shift + M                  | Command(⌘) + Shift(⇧) + M                    |

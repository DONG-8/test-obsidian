
### [[Part 2. 디자인 스타일 세팅]]
- [Style Settings](obsidian://show-plugin?id=obsidian-style-settings): CSS 지식 없이도 색상, 간격, 글꼴 등 테마 커스터마이징 지원
- [CSS Editor](obsidian://show-plugin?id=css-editor) 별도의 Editor 없이 CSS 코드 직접 편집 기능 지원 

### [[Part 3. 데일리노트 활용법]]
- [Calendar](obsidian://show-plugin?id=calendar): 캘린더 기반으로 일자별 노트 기록 시각화
- [ICS](obsidian://show-plugin?id=ics): 구글 캘린더 일정 가져오기 지원
- [Templater](obsidian://show-plugin?id=templater-obsidian): 반복적으로 사용하는 템플릿 노트 자동화
- [Outliner](obsidian://show-plugin?id=obsidian-outliner): 불렛 기반 노트 편집을 돕는 플러그인 (단축키 및 마우스 이동 지원)
- [Journal Review](obsidian://show-plugin?id=journal-review): 오늘을 기준으로 N일·N주·N월·N년 전 노트 자동 연결

### [[Part 4. 개인지식관리 방법론(PKM) 소개]]
- [Excalidraw](obsidian://show-plugin?id=obsidian-excalidraw-plugin): 자유로운 다이어그램 작성과 노트 연결이 가능한 시각화 도구

### [[Part 6. 독서 노트 관리]]
- [Korean Book Info](obsidian://show-plugin?id=kr-book-info-plugin): Yes24 도서 정보를 노트에 자동 삽입
- [Book Search](obsidian://show-plugin?id=obsidian-book-search-plugin): Google, Naver 도서 정보 가져오기 (API Key 필요)
- [List Callouts](obsidian://show-plugin?id=obsidian-list-callouts): 불렛 리스트를 강조하거나 하이라이트 표시
- [Callout Integrator](obsidian://show-plugin?id=callout-integrator): 여러 문장을 선택해 한 번에 콜아웃 형태로 감싸기
- [Callout Manager](obsidian://show-plugin?id=callout-manager): 콜아웃 자동 삽입 및 커스텀 서식 관리 지원

### [[Part 7. MOC(Maps of Content) 활용법]]
- [Dataview](obsidian://show-plugin?id=dataview): 쿼리 구문을 활용해 노트를 목록이나 표 형태로 동적으로 정리
- [Natural Language Dates](obsidian://show-plugin?id=nldates-obsidian): @today, @next week처럼 자연어로 날짜 입력
- [Auto Link Title](obsidian://show-plugin?id=obsidian-auto-link-title): URL 입력 시 페이지 제목을 자동으로 링크 텍스트로 삽입

### [[Part 11. 유용한 플러그인 소개]]
- [Smart Typography](obsidian://show-plugin?id=obsidian-smart-typography): 따옴표, 대시, 화살표 등 일반 기호를 보기 좋은 특수문자로 자동 변환
- [Paste image rename](obsidian://show-plugin?id=obsidian-paste-image-rename): 이미지를 붙여넣을 때 현재 노트 제목 기준으로 파일명 자동 변경
- [Recent Files](obsidian://show-plugin?id=recent-files-obsidian): 사이드바에 최근 열어본 노트 히스토리 표시
- [Dynamic Outline](obsidian://show-plugin?id=dynamic-outline): GitHub 스타일의 목차(TOC) 자동 생성. 노트 내 빠른 이동과 탐색 지원
- [OmniSearch](obsidian://show-plugin?id=omnisearch): 기본 검색 기능을 확장한 PDF, Office 문서, 이미지 OCR까지 전방위 검색 지원
- [Tag Wrangler](obsidian://show-plugin?id=tag-wrangler): 태그 패널에서 이름 변경, 병합, 검색 등 통합 태그 관리 기능 지원
- [Creases](obsidian://show-plugin?id=creases): 헤더나 제목의 접기/펼치기 상태를 기억하고 고정
- [Block Link Plus](obsidian://show-plugin?id=block-link-plus): 제목 뿐만이 아닌 텍스트 블록도 링크 및 embed 가능하도록 지원
- [Merge Notes](obsidian://show-plugin?id=merge-notes): 여러 노트를 하나의 통합 노트로 병합
- [Note Refactor](obsidian://show-plugin?id=note-refactor-obsidian): 긴 노트를 커서 위치, 제목 등으로 빠르게 분할
- [PDF++](obsidian://show-plugin?id=pdf-plus): PDF 파일과 노트의 유기적 연결(백링크, 하이라이트, 주석 기능 등)
- [Translate](obsidian://show-plugin?id=translate): 선택한 텍스트나 전체 노트를 즉시 번역하는 다국어 지원 플러그인
- [Smart Composer](obsidian://show-plugin?id=smart-composer):  Cursor AI 스타일의 AI 문서 작성 도우미
- [Smart Connections](obsidian://show-plugin?id=smart-connections): AI 기반으로 노트 간 의미적 유사성을 분석해 관련 노트 자동 추천(느림)
- [Share Note](obsidian://show-plugin?id=share-note): 작성한 노트를 테마와 포맷 그대로 유지한 채 외부 링크로 간편 공유
- [Custom Frames](obsidian://show-plugin?id=obsidian-custom-frames): 웹페이지를 사이드바 패널로 고정해 iframe으로 사용
- [Telegram Sync](obsidian://show-plugin?id=telegram-sync): 텔레그램 메시지와 파일을 옵시디언 보관함으로 자동 전송
- [YTranscript](obsidian://show-plugin?id=ytranscript): YouTube 동영상 스크립트를 1초 만에 추출해 타임스탬프와 함께 표시
- [Importer](obsidian://show-plugin?id=obsidian-importer): 다른 노트 툴(애플 노트,에버노트,노션, 원노트 등)에서 옵시디언으로 일괄 변환 및 가져오기
- [BRAT](obsidian://show-plugin?id=obsidian42-brat): 개발 중이거나 베타 버전 플러그인을 미리 설치하고 테스트할 수 있도록 지원

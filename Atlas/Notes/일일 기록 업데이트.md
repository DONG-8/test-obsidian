# Day 1 (월)
```
# Company
- 월간 회고 미팅 준비 (지난달 OKR 정리)
- 신규 기능 아이디어 브레인스토밍 회의 참석
- 경쟁사 리서치 자료 수집

- 독서: 《세컨드 브레인》 1장 읽기
- 옵시디언 셋업 (템플릿 폴더 정리)
- 주간 계획 수립

# Journaling
- [[오늘 내가 한 일 중 가장 잘한 것 한 가지]]
	- 팀 회의에서 내가 제안한 기능 아이디어가 긍정적인 반응을 얻었다. 사전 리서치가 효과를 발휘했다.
- [[오늘 내가 아쉬웠던 것 한 가지]]
	- 회의 후 피드백 내용을 바로 정리하지 않아 기억이 흐릿해졌다.
- [[내일 개선할 행동 한 가지]]
	- 회의 직후 10분 동안 회고 메모를 정리하는 루틴을 실천해보겠다.
```

# Day 2 (화)
```
# Company

	- 자동화 전에 문서 구조를 더 단순하게 정리하고 단계별로 테스트할 것이다.
```
